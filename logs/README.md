# Logs Directory

Store durable traces here.

Recommended files:

- `planner_trace.log`
- `generator_trace.log`
- `evaluator_trace.log`
- `playwright_trace.log`
- `test_output.log`
- `console_errors.log`

## Logging Rule

If a claim matters, log the evidence. If the evidence is not logged, the Evaluator should treat the claim as unsupported.
